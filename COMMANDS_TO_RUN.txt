RUN THESE COMMANDS IN ORDER:

1. Update remote to new repository:
   git remote set-url origin https://github.com/SyntaxisGlobal/Syntaxis.git

2. Create new orphan branch with clean history:
   git checkout --orphan clean_main

3. Add all files:
   git add .

4. Commit with correct message:
   git commit -m "Syntaxis Upload"

5. Delete old main branch and rename:
   git branch -D main
   git branch -m main

6. Force push to new repo:
   git push -f origin main

This will give you:
- Clean history with only one commit
- Commit message: "Syntaxis Upload"
- All dates will be August 2025 (current date)
- No unnecessary files
- No weird backdated commits
